## Clone & Initialisation:

```
git clone https://github.com/DjoAHP/MusicLib-ahp.git
npm install
npm run dev
```
## Push depot:
```
git add .

git push
```

## Cache chrome

Penser a modifier le nom de l app dans package.json et vider le cache Service Workers de chrome.
1. Ouvre Chrome sur http://localhost:5173/
2. Appuie sur F12 pour ouvrir les outils développeur
3. Va dans l'onglet Application (tu devras peut-être cliquer sur >> pour le trouver)
4. Dans le menu à gauche, clique sur Service Workers
5. Si tu vois un service worker listé, clique sur Unregister
